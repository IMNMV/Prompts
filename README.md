# Prompts

## I plan to update this repoistory for prompts I typically use in various workflows.

# Context Preservation is for any LLM to output a comprehensive explanation of everything that happened up until that point which can be fed into a fresh instance to pick up where you are leaving off from.

# Data Analysis is providing scaffolding rules for any LLM using data analysis. Currently the focus is on R. These are best practices I learned from my graduate statistic courses.
